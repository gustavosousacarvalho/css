Aulas css

CSS é a sigla para o termo em inglês Cascading Style Sheets que, traduzido para o português, significa Folha de Estilo em Cascatas. O CSS é fácil de aprender e entender e é facilmente utilizado com as linguagens de marcação HTML ou XHTML.

Seletores css

O seletor é a parte da regra CSS que está antes do sinal “{“ (chave de abertura). O seletor aqui é p , que "casa" com todos os elementos p do documento e faz com que qualquer texto dentro de um parágrafo seja na cor vermelha. Bem básico.

Seletor Descrição Exemplo
tag utilizado com o nome da tag para aplicar css 
p { background-color: gold;}
class utiliza o atributo class para estilizar o elemento 
.paragrafo{ background-color: gold;}
Universal usado para selecionar todos os elementos da pagina 
*{color:red}
Id usado para selecior um unico elemento 
#borda-tracejada{border: 2px doted }
